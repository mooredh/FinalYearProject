Sexual Predator Detector API

Hi! This API can take in chat data and determine if a sexual predator is in the chat. It will also tell you who the sexual predator is

Try it out by sending a post request to the path /api/v1/detect

Follow the below format for the request body:

        {
            "conversation_id": [{
                "author": String,
                "text": String
            }]
        }

Response would be in format:

        {
            "conversation_id": {
                "predator_detected": 0 or 1,
                "predator": String
            }
        }
